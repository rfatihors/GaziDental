# Roboflow YOLO Dataset

Place the exported Roboflow YOLOv8/YOLOv11 segmentation dataset here.

Expected structure:

```
roboflow_yolo/
├── data.yaml
├── images/
│   ├── train/
│   ├── val/
│   └── test/
└── labels/
    ├── train/
    ├── val/
    └── test/
```

`images/*` should contain the raw images and `labels/*` should contain YOLO-format
segmentation label files generated by Roboflow.
